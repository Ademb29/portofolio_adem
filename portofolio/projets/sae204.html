SAÉ 2.04 · BUT1 · Semestre 2

Visualisation
de données

Tableaux de bord sur les catastrophes climatiques par pays — cycle complet MCD → SQL → Python.

Le projet en bref

À partir d'un jeu de données réel sur les catastrophes climatiques par pays, j'ai mené tout le cycle complet : analyse des données brutes → modèle conceptuel (MCD) → modèle logique (MLD) → script SQL de création → import des données → visualisations Python.

Ce projet a combiné compétences en bases de données et en traitement de données avec Python (pandas, matplotlib), pour produire des visualisations pertinentes et lisibles.

Déroulement
Analyse et modélisation
Étude du jeu de données climatiques, identification des entités, conception du MCD et MLD.
Implémentation SQL
Création du schéma, import des données, requêtes d'agrégation pour préparer les données à visualiser.
Visualisations Python
Graphiques avec matplotlib et seaborn : évolution temporelle, comparaisons par région, corrélations.
La SAÉ qui m'a donné le goût de la data. Transformer des données brutes en graphiques qui racontent une histoire — les catastrophes climatiques par pays sur 50 ans — c'est à la fois technique et porteur de sens. Python + pandas, c'est devenu un de mes combos préférés.
