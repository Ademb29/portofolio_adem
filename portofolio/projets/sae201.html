SAÉ 2.01 · BUT1 · Semestre 2

Application Java
& Interface

Application Java orientée objet avec interface graphique et connexion à une base de données.

Le projet en bref

Développement d'une application Java complète avec interface graphique (JavaFX) et base de données. Montée en compétences sur l'architecture applicative en couches (UI, logique métier, accès données).

Ce projet a consolidé la POO apprise en S1 en l'appliquant à un contexte plus complet et plus réaliste, avec des contraintes d'interface et de persistance des données.

Déroulement
Architecture de l'application
Séparation des couches : présentation (JavaFX), logique métier, accès aux données (JDBC).
Développement de l'interface
Création des vues avec JavaFX, gestion des événements, navigation entre les écrans.
Intégration base de données
Connexion SQL, requêtes CRUD, gestion des erreurs et transactions.
La première fois que j'ai vu mon interface graphique interagir avec une base de données en temps réel, j'ai compris l'intérêt de la séparation des couches. Ce projet a donné du sens à tout ce qu'on avait appris séparément en S1.
