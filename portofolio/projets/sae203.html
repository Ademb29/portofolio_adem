SAÉ 2.03 · BUT1 · Semestre 2

Administration
système Linux

Installation et configuration d'un serveur LAMP complet sur machine virtuelle Debian.

Le projet en bref

Première vraie incursion dans l'administration système. Installation d'une machine virtuelle Debian sous VirtualBox, puis configuration d'un serveur LAMP complet : Linux, Apache (serveur web), MySQL (base de données), PHP (langage côté serveur).

L'objectif : avoir un environnement de développement web fonctionnel, comme dans une vraie entreprise.

Déroulement
Installation de la VM Debian
Configuration de VirtualBox, installation Debian, partitionnement du disque, réseau NAT.
Configuration LAMP
Installation et configuration d'Apache, MySQL et PHP. Premiers scripts PHP connectés à la base.
Déploiement et tests
Déploiement d'une application web simple sur le serveur. Vérification du bon fonctionnement complet.
Comprendre comment fonctionne un serveur web de l'intérieur change la façon de développer. Je ne vois plus un site web de la même façon — je pense maintenant à Apache, aux droits des fichiers, à la configuration PHP. Cette SAÉ m'a donné envie de comprendre l'infrastructure.
